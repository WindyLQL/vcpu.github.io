strace跟踪程序执行时间

发表于 2018-01-13 | 分类于 linux kernel | | 阅读次数

strace可以帮助你打印函数执行时间，帮助你查看程序消耗，针对于复杂的工作进程例如nginx或者janus，我暂且没有尝试，但是对复杂程序性能调整作用不大。

遇到复杂程序你可以考虑valgrind –tool=callgrind工具
